# Age of Sigmar App

## Installation

#### 1. Make sure you have expo-cli installed globally

`npm install -g expo-cli`

#### 2. Make sure you have yarn installed globally

`npm install -g yarn`

#### 3. Install packages

`yarn`

## Usage

#### 1. Run the project

`yarn start`
